# 활동 내용

### 객체 지향의 3가지 
- 협력
- 역할 
	- (역할이 수행해야 할 책임을 다 할 수 있다면 해당 역할을 수행할 수 있다. 즉 구현체가 바뀌더라도 역할을 수행할 수 있다면 문제가 없다. )
	- e.g.1. 자동차 역할, 운전자 역할
	- e.g.2. 공연무대(협력). 로미오 역할, 줄리엣 역할
	- e.g.3. 카페(협력). 주문자 역할, 카페 알바 역할
- 책임

### 코드 리뷰
